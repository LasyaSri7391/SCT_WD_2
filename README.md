# ⏱️ Stopwatch Web Application

A modern and responsive stopwatch web application built using **HTML**, **CSS**, and **JavaScript**. The application allows users to accurately measure elapsed time, pause and resume the stopwatch, reset it, and record lap times through a clean and intuitive interface.

## 📌 Features

- Start, Pause, Resume, and Reset functionality
- Millisecond precision timing
- Record and view lap times
- Live lap counter
- Status indicator (Ready, Running, Paused)
- Responsive design for desktop and mobile devices
- Smooth animations and modern dark-themed UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
